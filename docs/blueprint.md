# **App Name**: Viby

## Core Features:

- Project Rollback Tool: An AI tool that reasons through the project history to identify and remove all PublicLens modules, restoring the original Viby codebase structure and fixing build errors caused by missing exports.
- Hard Lock Governance Tool: An AI tool that reasons about user commands to enforce a Kill Switch on all autonomous processes, ensuring the system operates solely in Direct Edit Mode with zero planning or scaffolding.
- Strict Manual Execution Tool: An AI tool that reasons about incoming text instructions to execute them literally, such as updating imports from useDocument to useDoc to resolve compilation errors, without any autonomous planning.
- Admin-Linked Logo Tool: An AI tool that reasons how to dynamically resolve and fetch the site logo based on definitions found in the /admin/configuracoes path.
- Public Navigation Hub: A dedicated UI component containing navigation links for /copa-do-mundo and /experiencias.
- Auth Management Header: A comprehensive UI header interface for registration, login, profile management, and secure logout.

## Style Guidelines:

- Primary Color: Vivid Indigo (#3D5AFE) for core interactive elements.
- Background Color: Pale Mist (#F5F7FF) for a clean, clinical feel.
- Accent Color: Electric Cyan (#00B0FF) for tooltips and highlights.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, geometric tech feel.
- Body font: 'Inter' (sans-serif) for high legibility in navigation labels.
- Code font: 'Source Code Pro' (monospace) for technical file paths.
- A dual-column sidebar layout featuring a 'Master-Detail' view for exploring project files.
- Fluid, low-latency micro-interactions that trigger when file paths are expanded or buttons are interacted with.
