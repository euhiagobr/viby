# **App Name**: Viby

## Core Features:

- Mural Público de Eventos: Lista de cards visualmente impactantes para descoberta rápida de eventos públicos e privados.
- Filtros e Busca por Localidade: Mecanismo de pesquisa rápida baseado em cidade e data para facilitar o encontro de experiências.
- Gestão Centralizada de Eventos: Interface para organizadores criarem eventos com detalhes completos como local, lotes e valores armazenados via Firestore.
- Assistente de Marketing IA: Uma ferramenta de IA que atua como copiloto para gerar propostas comerciais e descrições criativas para os eventos.
- Controle de Lotes Dinâmicos: Gerenciamento completo de disponibilidade e preços de ingressos com persistência no banco de dados.
- Mapa Interativo de Experiências: Visualização geográfica integrada para localizar eventos e pontos de interesse em tempo real.
- Dashboard Kanban para Promoters: Organização do status da produção do evento (planejado, em progresso, concluído) com visualização Kanban.

## Style Guidelines:

- Cores Principais: Esquema de cores focado em modo escuro elegante com preto profundo (#000000) e o azul vibrante (#007bff) como sotaque técnico.
- Fundo: Tom desaturado de preto-azulado (#0a0f1a) para garantir contraste suave com as bordas e cards.
- Fontes Recomendadas: Utilização de 'Poppins' para títulos (Sans-serif geométrico) e corpo de texto, oferecendo uma leitura precisa e contemporânea. Note: atualmente apenas Google Fonts são suportadas.
- Ícones: Estilo minimalista com traços finos (line icons) que utilizam o sotaque azul para indicar estados ativos ou interação.
- Arquitetura: Design limpo com vasto uso de espaços em branco (negative space), utilizando elevação via sombras leves em cards para criar profundidade.
- Interações: Transições suaves de entrada e saída (200-300ms) para mudanças de filtros e checkouts animados e satisfatórios.
